# twitter-thread

将长文/文稿转换为 Twitter 帖子串的 Claude Skill

## 描述

把想法或素材变成一串能抓人、读得下去、有落点的 X/Twitter thread。

核心信念：**好的 thread 是用"钩子 + 干货 + 案例 + 观点 + CTA"五种积木搭出来的。**

## 功能

- 把长文章转换为 Twitter 帖子串
- 自动分段（5 种积木框架）
- 优化推文格式
- 中文/英文双语支持

## 触发词

- "帮我写个 thread / 写一串"
- "这个想法/文章/笔记,改成 thread"
- "我要发一串介绍 XX"

## 核心工作流

1. **第一条给极强的钩子**——抓住注意力
2. **中段给有价值的信息或故事**——让读者有收获
3. **核心亮点自然呈现**——不硬推

## 使用方法

1. 安装 skill 到 Claude/Cursor/Windsurf
2. 输入长文章或素材
3. AI 自动生成 Thread

## 依赖

- Claude (或其他 AI)
- 网络访问
