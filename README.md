# twitter-thread

把想法变成一串能抓人、读得下去、有落点的 Twitter 帖子。

## 核心信念

好的 thread = 钩子 + 干货 + 案例 + 观点 + CTA

## 功能

- 生成抓人眼球的 Thread
- 自动分段（5 种积木框架）
- 优化推文格式
- 中英双语支持

## 触发词

- "写个 thread"
- "改成 thread"
- "发一串"

## 使用方法

1. 安装到 Claude/Cursor/Windsurf
2. 说出你的想法或粘贴素材
3. 自动生成 Thread

## 依赖

- Claude (或其他 AI)
- 网络访问
